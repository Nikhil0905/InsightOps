                INTELLIGENCE
                
Risk Scoring Formula

                
Fully deterministic. Fully explainable. No black boxes.

            

            

                

                    

                        
Base Score Calculation

                        

                            
risk_score = normalize(
    severity       × w₁ +
    host_crit      × w₂ +
    user_priv      × w₃ +
    event_freq     × w₄
) + correlation_bonus

→ capped at 100
                            
                        

                    

                    

                        
Default Weights (config/weights.yaml)

                        

                            

                                Base Severity
                                

                                    

                                

                                1.0
                            

                            

                                Host Criticality
                                

                                    

                                

                                1.0
                            

                            

                                User Privilege
                                

                                    

                                

                                1.0
                            

                            

                                Behavioral Freq
                                

                                    

                                

                                1.0
                            

                            

                                Correlation Bonus

                GET STARTED
                
Quick Setup

                
Get InsightOps running in under 5 minutes.

            

            

                

                    
01

                    

                        
Install Dependencies

                        

                            

                                bash
                                
                            

                            
pip install -r requirements.txt

                        

                    

                

                

                    
02

                    

                        
Configure Credentials

                        

                            

                                bash
                                
                            

                            
cp .env.example .env
# Edit .env with your values:
#   SPLUNK_USERNAME=...
#   SPLUNK_PASSWORD=...
#   SPLUNK_HEC_TOKEN=...
source .env

                        

                    

                

                

                    
03

                    

                        
Configure Connection

                        

                            

                                yaml — config/splunk.yaml
                                
                            

                            
splunk_host: 10.10.10.1
management_port: 8089
hec_port: 8088
ai_soc_index: ai_soc

                        

                    

                

                

                    
04

                    

                        
Run InsightOps

                        

                            

                                bash
                                
                            

                            
# Dry-run (no writes to Splunk)
python ai-engine/main.py --dry-run

# Full pipeline
python ai-engine/main.py
